Objectives

To make a basic C++ program incorporating variables and strings.
To use cout to display student information.
To structure the program to get comments to make it easier to read.
Practice writing, compiling and running a program in C++.
To utilize Git to monitor the changes in the project.
To upload the project to GitHub using Git commands.
To be able to grasp the underlying concepts of Git and GitHub's version control system.

Outcomes

Successfully created and executed the C++ program without errors.
Being able to declare variables and store string data.
Drew information that had been formatted, as it would appear to the student, onto the console.
Added comments and formatting for improved coding style.
Used Git commands to manage and track the project files.
Successfully uploaded the project to GitHub.
Learned about the Git and GitHub workflow, such as adding files, making commits and pushing them to a remote repository.
Gained confidence in implementing version control in future programs.
