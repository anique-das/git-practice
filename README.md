Objectives

To create a simple C++ program using variables and strings.
To display student information using cout.
To organize the program with comments for better readability.
To practice writing, compiling, and running a C++ program.
To use Git for tracking changes in the project.
To upload the project to GitHub using Git commands.
To understand the basic workflow of Git and GitHub for version control.

Outcomes

Successfully created and executed the C++ program without errors.
Learned how to declare variables and store string data.
Displayed formatted student information on the console.
Improved coding style by using comments and proper formatting.
Used Git commands to manage and track the project files.
Uploaded the project successfully to GitHub.
Gained practical knowledge of the Git and GitHub workflow, including adding files, committing changes, and pushing them to a remote repository.
Developed confidence in using version control for future programming projects.
